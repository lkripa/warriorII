## Installation

```
$ git clone https://github.com/lkripa/warriorII.git
$ cd warriorII
$ pod install
$ curl -o SwiftOpenPose/Resources/MobileOpenPose.mlmodel \
https://s3-ap-northeast-1.amazonaws.com/swiftopenpose/MobileOpenPose.mlmodel
```
